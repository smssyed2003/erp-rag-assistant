```
                       ~$0.01-0.10 per 1000 requests (beyond free)
─────────────────────────────────
Total:                 $0 - $5/month
```

### When to Upgrade
```
Render → $7/month when:
  - You want always-on backend
  - You have >15 minute quiet periods

Vercel → ~$20/month when:
  - You need more resources
  - You scale to 100k+ users

Google Gemini → when:
  - Free tier exhausted (~1000 requests/month)
  - Then ~$0.0001-$0.0004 per request
```

## 📞 SUPPORT & TROUBLESHOOTING

### Common Questions

**Q: Which file should I read first?**
A: `EXECUTION_GUIDE.md` (if deploying now) or `docs/ERP_RAG_Complete_Guide.md` (if learning)

**Q: How long until backend responds?**
A: 30-40 seconds on free tier (it sleeps). Click again after waiting.

**Q: Can I modify the code?**
A: Yes! Everything is open source. Modify `backend/app/rag_engine.py` to change prompts.

**Q: How do I add more ERP documents?**
A: Edit `backend/data/erp_chunks.json` and add more dictionaries to the list.

**Q: Is my API key secure?**
A: Yes. It's in `.env` file (not pushed to GitHub - check `.gitignore`).

**Q: Can I run this locally forever without deploying?**
A: Yes! Just run locally. Deployment is optional, but fun to share!
